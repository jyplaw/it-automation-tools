# IT Automation Tools

This project contains simple automation tools built using Python.

## Features

- Server Monitoring (CPU, Memory, Disk)
- Log Analyzer (Detect ERROR and WARNING)
- Auto Backup Script (Zip with timestamp)
- Alert System (Print / Email alert)

## Technologies

- Python
- psutil
- shutil
- smtplib

## How to Run

```bash
python server_monitor.py
